# Personalized Content Dashboard

For detailed setup and usage instructions, see [README.md](./README.md).

## Quick Start

1. Install dependencies: `npm install`
2. Copy `.env.local.example` to `.env.local` and add your API keys
3. Run development server: `npm run dev`
4. Open http://localhost:3000

## Key Commands

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run test` - Run unit tests
- `npm run test:e2e` - Run E2E tests
- `npm run lint` - Lint code

## Features

✅ Multi-source content aggregation (News, Movies, Social)
✅ Category filtering and search
✅ Infinite scrolling
✅ Favorites with drag-and-drop reordering
✅ Dark/light mode toggle
✅ Fully responsive
✅ Comprehensive testing
✅ TypeScript + Redux Toolkit + RTK Query
